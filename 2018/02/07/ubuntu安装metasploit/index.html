ubuntu 安装 metasploit

metasploit 是一款老牌的漏洞利用工具了。以前总是在windows上面安装它。或者直接使用kali的版本。没想过在linux上面安装。今天有需求，就试着安装了 一下。比windows下面使用的爽多了。
下面是安装步骤

  1. 打开linux console终端
  2. 下载安装程序：
    $ wget http://downloads.metasploit.com/data/releases/metasploit-latest-linux-x64-installer.run
    根据自己的版本选择x64还是x32
  3. 添加文件的执行的权限
    user@ubuntu:~4 chmod +x ./metasploit-latest-linux-x64-installer.run
  4. 运行安装程序
    sudo ./metasploit-latest-linux-x64-installer.run
  5. 然后一直回车，中间有个Y要输入。

  6. 不要忘记注册一个用户！API需要用户登录
